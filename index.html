Application Management System

.Net Framework MVC C# CRUD Application
The application manages user applications by performing basic CRUD(Create,Read,Update,Delete) operations.
The list of applications is displayed in the landing page. The user can search a specific application by its title.
A new application can be added by the user by filling the required data in the form. The data is validated and specific error messages are displayed to the user if the data is incomplete/incorrect. A document can be uploaded in the form and can be downloaded later by clicking the "Download" image-button.
The user can also edit or delete the application(he must confirm his action before deleting).

Tech-Term

Tech-Term is an electronic dictionary, which contains engineering and technology related terms.
Any site visitor can search and read terms from the dictionary.
Students of Information Technology Faculty,who own an official e-mail address(domain: @fti.edu.al) can register and contribute by adding new terms and concepts to the dictionary. They can also save concepts as "Favorites".
The system has an administrator, who takes care of inappropriate/incorrect terms or explanations, by deleting/editing them.
Other functionalities are demonstrated in the demo.
Tech-Term was developed using PHP,JavaScript,CSS and HTML.

Yahtzee Game

Yahtzee Game-Desktop application developed with Java Swing (GUI)
The Yahtzee scorecard contains 13 different categories and in each round, after the third roll, the player must choose one of these categories. The score entered in the box depends on how well the five dice match the scoring rule for the category.
